🔗 FastAPI Webhook — POST /hitl/decision
Reçoit : { thread_id, decision: "approve"|"modify"|"block", notes? }
Appelle : graph.invoke(None, config={"thread_id": thread_id})
Résultat : thread LangGraph reprend exactement là où il était suspendu
⏱ Timeout Handler — 4h sans réponse
Si le chef ne répond pas dans 4h :
→ status: "timeout" injecté dans SharedState
→ supervisor_node escalade · thread relancé automatiquement
N1 — Automatique
Aucune approbation requise. Relances 1er niveau, rapports standard, alertes routine.
N2 — Notification
Agent agit + notifie. Chef peut annuler dans la fenêtre définie. Faible risque.
N3 — Approbation ⏸
Agent prépare, chef décide avant action. Écarts réconciliation, escalades clients, fiscal.
N4 — Transfert humain
Cas hors règles connues. Agent documente et transfère complètement au comptable.
