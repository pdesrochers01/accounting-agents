# AccountingAgents

Multi-agent LLM framework for accounting firm automation.

> **Status**: Pre-release. Paper under preparation for arXiv submission.
> Repository will be made public upon paper release.

## Overview

AccountingAgents is a multi-agent LLM framework inspired by the organizational
structure of professional accounting firms, built on LangGraph and MCP-native
tool integrations.
